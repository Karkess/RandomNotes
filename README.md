# RandomNotes
Notes that I wrote on a whim that offered to be uploaded to GitHub. 

This has been a collection of digital notes I've typed since roughly October 2025. They aren't anything other than a brief summary of what I've read in a book or paper sometimes nearly word for word. I type them to force me to slow down and more carefully read what I've been written. I actually take even shorter notes by hand and do practice problems by hand but toss them, so this is just one part of three that I am keeping just so I can look back on what I've done. For any review purposes I'd propose going to the original material, or if you're already familiar with the original material just check out a new source, the second teacher is always better.
